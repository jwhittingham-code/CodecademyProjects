Html-CSS Cheatsheet

Things I forget

Tables

<table> tags create a blank table

<thead> is semantic html to group header elements
<th> define headers per row <tr> tags create a new row in the table <td> tags create a new cell within that row
<th> are also used in <thead> using colspan in <td> will create a cell that spans a specified number of columns, in this case two
rowspan does the same for rows, again in this case two
<tfoot> is semantic html to group footer elements
This is a <caption>, these can be added inside of <table> captions are added to the top of the table by default

Css Flexbox

to define an element and its children with flexbox we use the display element in CSS

#item { display: flex;}
